GetErrorText Method
Class: FTP_SERVICE
This method is used internally, but seems like it might be useful as an exposed function. If a unknown error code is passed, this method just returns "Unknown Error Message" .
RetVal 
=
Object.GetErrorText
(lnError)
Parameter
lnError
Error number code.
Type Numeric
Direction Input
An error code returned by GetErrorCode()
Return value Character
Returns a text message relating to the error code passed.
Example 
LOCAL loFTP,lcPom
#INCLUDE "ftp.h"
SET PROCEDURE TO ftp.prg ADDITIVE 
loFTP=CREATEOBJECT('ftp_service') 

IF loFTP.OpenInternet("ABONNE", "PWD", "10.10.10.10", "21")
   IF !loFTP.ChangeFTPDirectory("_!_"+CHR(1)+CHR(9))
      loFTP.GetErrorText(This.GetErrorCode())
   ENDIF
   =loFTP.CloseInternet() 
ENDIF
RELEASE PROCEDURE ftp.prg
