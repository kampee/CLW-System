GetCachingMethod Method
Class: FTP_SERVICE
Returns the current caching method.
RetVal 
=
Object.GetCachingMethod
()
Return value Numeric
The return value is the current caching method - value of property nCachingType.
Example 
#INCLUDE "ftp.h"
LOCAL loFTP
SET PROCEDURE TO ftp.prg ADDITIVE 
loFTP=CREATEOBJECT('ftp_service') 
IF loFTP.SetCachingMethod(INTERNET_FLAG_DONT_CACHE)
   ?loFTP.GetCachingMethod() 
ENDIF
RELEASE PROCEDURE ftp.prg

FTP_SERVICE::SetCachingMethod() | FTP_SERVICE::ResetCachingMethod() | nCachingType
